what are partials?

partials are a block of repetative code that you write once in a file and use <%-include("/* file location*/") %> to attach in 
every other file where the code repeates. It saves time and reduces repeattive code, easier debugging. 

these are important things for me to remember:-

1) <%= %> --> variable 
2) <%- %> --> html code / attach partials using include.  
3) <% %> --> JS code / show ejs. 
4) <%# %> --> comments
