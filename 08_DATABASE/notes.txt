What is CRUD?

CRUD stands for 
C - Create
R - Read 
U - Update 
D - Delete

SQl commands for basic CRUD Operations.

1) Create tables

CREATE TABLE STUDENT (
    roll_no int,
    name varchar(20),
    class char(1),
    PRIMARY KEY(id)
);

2) INSERT data

DML commands


1) SELECT command:- 

// select all from table.

SELECT * FROM Student;

// select single/mutliple fields from table. 

SELECT id, name FROM STUDENT;

// select a field/ fields from table with a ceratin condition. 

SELECT id, name 
FROM STUDENT
WHERE (id == 1);

2) UPDATE commands

UPDATE STUDENT 
SET id = 4, class = 'C'
WHERE (id == 2)

3) DELETE command 

DELETE FROM STUDENT
WHERE (id == 3)

4) DROP command

DROP TABLE STUDENT

FOREIGN KEYS:- 

CREATE TABLE PRODUCTS (
    id SERIAL,
    customer_id INT, 
    product_id INT, 
    PRIMARY KEY(id),
    FOREIGN KEY (customer_id) REFERENCE customers(id),
    FOREIGN KEY (product_id) REFERENCE products(id)
);
