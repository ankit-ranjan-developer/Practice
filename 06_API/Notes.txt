what is axios? 

Axios is a popular npm package used for handling the APIs. 
We use axios to make the request and get the response from an API, It has defined format to make API request and we get the response 
as JS object. we don't need to even parse it, Axios does everything in the background for us. 

Then I have also studied about what is API authentication. we have four different types of API authentication. They are as follows:-

1) NP AUTH (but maybe have limited calls to reduce abuse and exploitation of teh API)
2) BASIC authentication, uses the username and password. 
3) API key, use the API key that is generated by the server to make request. 
4) Token 

In the first practice program, I have used all the four types of authentication on my web app. I used the API "https://secrets-api.appbrewery.com/"
to make my project. I learnt about all the different types of API authentication.
